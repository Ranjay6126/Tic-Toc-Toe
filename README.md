# Tic-Toc-Toe
Tic-Toc-Toe using JavaScript 

Tic-Tac-Toe Game (JavaScript)

This is a simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The project demonstrates basic game logic, DOM manipulation, and event handling in JavaScript.

Players take turns marking X and O on a 3Ã—3 grid. The game automatically checks for a winner or a draw and displays the result.

Features

Two-player gameplay (X and O)

Win detection logic

Draw detection

Restart / Reset game option

Interactive user interface

Simple and clean design

Technologies Used

HTML

CSS

JavaScript

How It Works

Players click on empty boxes to place their mark (X or O).

The game checks for a winning combination after each move.

If all boxes are filled without a winner, the game ends in a draw.

Players can reset the game and play again.

Purpose

This project is created to practice JavaScript fundamentals such as functions, conditions, arrays, and DOM manipulation while building a fun and interactive game.
